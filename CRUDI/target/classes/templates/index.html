Proyecto Final de Arquitectura de Software

Este proyecto es una aplicación con operaciones CRUD desarrollada con Spring Boot y MySQL. Permite gestionar información de personas, incluyendo su registro, actualización, consulta y eliminación.

La aplicación tiene como objetivo consolidar los conocimientos adquiridos durante el curso y proporcionar una solución práctica a la gestión de información en sistemas modernos.

Desarrollado como parte de la materia Arquitectura de Software, 2024.
